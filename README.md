# Malinao Water Billing App

A Flutter application for managing water billing, customers, and payments with Firebase backend integration.

## Installation

1. Clone the repository:

```
git clone https://github.com/yourusername/malinao-water-billing.git
cd malinao-water-billing
```

2. Install dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```
